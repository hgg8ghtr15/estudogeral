## atributo blobais
--> id="nome" = RG, não repete,  seguir padrão de variável na nomenclatura

--> class="produto" = usado para CSS e JS, pode usar varias
<div class="produto calcado"> </div>

--> data-id = usado dentro do CSS ou JS
<div data-qualquer-coisa="teste"> </div>

--> style = usada para CSS, use pouco
<div style="color:red"> </div>
